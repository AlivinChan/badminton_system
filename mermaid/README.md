# Mermaid 图表说明

本文件夹包含了校园羽毛球馆场地预约管理系统的各种架构图和流程图，使用 Mermaid 语法编写。

## 图表列表

### 1. 系统架构图 (`01-系统架构图.md`)
展示了系统的整体架构，包括UI层、服务层、持久化层、模型层和工具层之间的关系。

### 2. 类图 (`02-类图.md`)
UML类图，展示了所有类的属性、方法和类之间的关系。

### 3. 数据流图 (`03-数据流图.md`)
展示了数据在系统中的流动过程，从用户操作到数据存储的完整流程。

### 4. 预约流程时序图 (`04-预约流程时序图.md`)
详细展示了学生创建预约的完整时序，包括冲突检测、费用计算等步骤。

### 5. 登录流程时序图 (`05-登录流程时序图.md`)
展示了学生注册、学生登录和管理员登录的完整流程。

### 6. 数据模型关系图 (`06-数据模型关系图.md`)
ER图，展示了实体之间的关系，包括学生、管理员、场地、预约等。

### 7. 组件依赖图 (`07-组件依赖图.md`)
展示了各个组件之间的依赖关系，清晰展示了系统的模块化设计。

### 8. 状态转换图 (`08-状态转换图.md`)
展示了程序状态、预约状态和场地状态的转换过程。

### 9. JSON解析流程图 (`09-JSON解析流程图.md`)
详细展示了自定义JSON解析算法的执行流程，包括大括号匹配算法。

### 10. 冲突检测算法流程图 (`10-冲突检测算法流程图.md`)
展示了时间段冲突检测的完整算法流程。

### 11. 费用计算流程图 (`11-费用计算流程图.md`)
展示了费用计算的决策流程，包括工作日/周末、白天/晚间的判断。

## 如何使用

### 在线查看
1. 将 `.md` 文件内容复制到支持 Mermaid 的在线编辑器：
   - [Mermaid Live Editor](https://mermaid.live/)
   - [GitHub](https://github.com) (直接查看 `.md` 文件)
   - [GitLab](https://gitlab.com) (直接查看 `.md` 文件)

### 在 Markdown 编辑器中查看
支持 Mermaid 的编辑器：
- VS Code (安装 Mermaid 插件)
- Typora
- Obsidian
- Notion

### 导出为图片
1. 使用 Mermaid Live Editor 打开文件
2. 点击 "Actions" -> "Download PNG/SVG"

### 在文档中引用
可以在项目文档中直接引用这些图表：

```markdown
详见 [系统架构图](mermaid/01-系统架构图.md)
```

## 图表说明

### 颜色说明
- **蓝色**: UI层组件
- **橙色**: 服务层组件
- **绿色**: 持久化层组件
- **紫色**: 模型层组件
- **黄色**: 工具层组件
- **红色**: 数据文件

### 图表类型说明
- **流程图 (flowchart)**: 展示流程和决策
- **时序图 (sequenceDiagram)**: 展示对象间的交互顺序
- **类图 (classDiagram)**: 展示类的结构和关系
- **ER图 (erDiagram)**: 展示数据模型关系
- **状态图 (stateDiagram)**: 展示状态转换
- **依赖图 (graph)**: 展示组件依赖关系

## 更新说明

1. 修改对应的 `.md` 文件
2. 确保 Mermaid 语法正确
3. 在 Mermaid Live Editor 中验证
4. 更新本 README 文件（如需要）
